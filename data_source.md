## Data Source

**Dataset:** Assembled Workers' Compensation Claims: Beginning 2000
**Published by:** New York State Workers' Compensation Board via data.ny.gov
**Last Updated:** March 23, 2026

🔗 [Download the dataset](https://data.ny.gov/Government-Finance/Assembled-Workers-Compensation-Claims-Beginning-20/jshw-gkgu/about_data)

> Note: The dataset contains over 1 million records and is too large to store in this repository. Please download it directly from the link above and save it to this folder as `Assembled_Workers__Compensation_Claims__Beginning_2000.csv` before running the notebooks.
